Live Sync Setup Guide

This method turns your Google Sheet into a Live Database. Your dashboard will automatically sync and update without writing a single line of code!

1
Open your Google SheetGo to the Google Sheet where your data is stored. Ensure your data is formatted correctly (Row 1 must contain column headers/names).
2
Click File > Share > Publish to webFrom the top menu, click on 'File', hover over 'Share', and select 'Publish to web'.
3
Change format to CSV CRITICAL STEPIn the popup window, change the default 'Web page' option to Comma-separated values (.csv).
If you do not select CSV, the dashboard will fail to read the data correctly.
4
Publish & Copy LinkClick the 'Publish' button. A link will be generated. Copy that link, paste it into the URL box in the previous window, and hit Connect!

--- HOW TO SETUP OWN GITHUB (5 STEPS) ---
1
GitHub Username (Owner)Your 'Owner' name is simply your GitHub Username.
Example: If your profile link is github.com/john-doe, then Username is john-doe.
2
Create Repository (Repo Name)Click '+' (Top Right) on GitHub > 'New repository'. Enter a name (e.g., 'my-dashboard').
IMPORTANT Select PUBLIC and check Add a README file. Click Create.
3
Branch NameBy default, GitHub creates a branch named main. It is already pre-filled. Do not change it.
4
Create Live Link (Base URL)In your Repo: Go to Settings > Pages (left sidebar). Under Build/Deployment, select Branch: main and click SAVE. Wait 1 min, refresh, and copy the link 'Your site is live at https://...'
5
GitHub Token (Access Password) Click here to open Token Page
Name it 'Dashboard Upload'. Set Expiration to 'No expiration'.
CRITICAL Check the box named 'repo' (Full control). Click Generate and copy the ghp_... code.
